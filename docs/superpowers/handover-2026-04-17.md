# OpenDNA — Session Handover, 2026-04-17

Written at the end of the MVP build session to orient a future Claude Code session that picks up this repo cold.

## What this is

Local-first web utility that parses raw 23andMe-style DNA files, matches them against 8 curated SNP panels, joins findings with ClinVar + PharmGKB/CPIC annotations, and renders a self-contained HTML report. Optional BYOK LLM synthesis (Anthropic default with prompt caching, OpenAI secondary) adds a prose interpretation layer. Everything runs on `localhost:8787`.

Seeded from a personal genomics sandbox at `~/Coding/sandbox/` (hand-curated panels + a static synthesis report). The sandbox is NOT a dependency — just the original inspiration.

## Key design decisions (for context)

| Decision | Choice | Rationale |
|---|---|---|
| Backend | FastAPI + Uvicorn | Python-native, batteries-included for JSON APIs |
| Frontend | Vanilla JS, no build step | Zero contribution friction, no npm |
| Privacy | API keys per-request only, never persisted | Trust claim verifiable by reading 50 LOC |
| Annotations | Shipped subsets + `update-db` | Instant first-run, explicit refresh path |
| LLM | Optional, pluggable, BYOK | Rule-based works without any key |
| Default model | `claude-sonnet-4-6` + prompt caching | Matches user's existing tooling |
| License | MIT | Max adoption for open-source |
| Port | 8787 | Unused by common dev stacks |
